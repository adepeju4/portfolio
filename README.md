# Portfolio

Personal portfolio site built with **SvelteKit**, **Tailwind CSS**, and **Sanity CMS**.

## Stack

- [SvelteKit](https://kit.svelte.dev) — framework
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Sanity](https://sanity.io) — content management

## Dev

```bash
pnpm install
pnpm dev
```

## CMS

```bash
pnpm studio
```

## Build

```bash
pnpm build
```
